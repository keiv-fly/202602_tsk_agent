# Scriber

Scriber is a Playwright-based browser session recorder. This initial scaffolding validates that the tool can start inside the configured environment.

## Requirements

- Node.js 20+
- npm

## Setup

```bash
cd scriber
npm install
```

## Run (record placeholder)

```bash
npm run record
```

Set `HEADLESS=false` to launch a visible browser window:

```bash
HEADLESS=false npm run record
```

## Tests

```bash
npm test
```
